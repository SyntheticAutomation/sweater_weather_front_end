## Design/Features
This project uses the following design principles & strategies:

- CSS
    - 7-1 Architecture
    - SCSS Syntax
    - Easing Functions
    - Custom Grids
    - Custom Animations

- HTML
    - BEM (Block Element Modifier) Tag Conventions

- JavaScript
    - Event Bubbling
